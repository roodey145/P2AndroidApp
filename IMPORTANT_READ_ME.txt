To find the html, css, js files go to app>src>main>assets.
The application will work on a normal website, however, to see how it will looks on a mobile phone. After going to the html folder and 
opening the index.html file, left click on the browser window and select inspect or click ctrl+shift+i a window will appear.
At the top-left conrnor of that window there is a mobile icon click on it to activate the mobile view.

To build the application and test it on a phone, you need to download Android Studio.


A ready build apk version can be found inside the apk folder.
